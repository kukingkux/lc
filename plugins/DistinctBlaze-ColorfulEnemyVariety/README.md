# Colorful Enemy Variety

### ColorfulEnemyVariety is a mod that adds over 100+ different recolor skins to the vanilla entity roster of Lethal Company in a effort to add simple variety to the enemies you see, similar to how Tulip Snakes have alternate colorations seen in Vanilla.

![](https://i.imgur.com/ZQ841XZ.png)

Featuring countless enemy colors for practically every entity in the game (besides Masked) and EnemySkinRegistry configuration, you can immerse yourself with skins to mix things up or better suit certain enemies on certain moons.

### If you want more unique & fun skin variety for the entities, check out [ExtraEnemyVariety](https://thunderstore.io/c/lethal-company/p/DistinctBlaze/ExtraEnemyVariety/)!

![](https://i.imgur.com/0eo7yAm.png)

# CONFIGURATION (ENEMYSKINREGISTRY)

Through EnemySkinRegistry (a dependency for EnemySkinKit), you can configure all the skins within the registry's config sub-menu, seen in-game.

![](https://i.imgur.com/edLQx4C.png)
*(Requires LethalConfig to view)*

Configurations include:
- <b>Enabling/Disabling individual skins</b>
- <b>General skin spawn chances (including Vanilla)</b>
- <b>Moon-specific skin spawn chances (including all LLL moons and moon tags)</b>

# NOTES & TROUBLESHOOTING

- <b>Do not edit EnemySkinRegistry's skin config file directly.</b> It's extremely specific and can cause errors. It is intended to configure through LethalConfig only.

- The Indoor/Outdoor config option in EnemySkinRegistry is a <i>bit</i> buggy and can lead to issues down the road if enabled then disabled. Caution is advised using that feature.

- <b>Vanilla skins are set to 0% selection chance by default in EnemySkinRegistry.</b> Be sure to configure it if you still want normal enemy skins in play.

- To enable skin syncing across players, have the host enable "AllowSync" and clients enable "AttemptSync" within EnemySkinRegistry's config.

- If skins don't seem to load, make sure you have [FixPluginTypesSerialization](https://thunderstore.io/c/lethal-company/p/Evaisa/FixPluginTypesSerialization/) as it's needed for EnemySkinRegistry.

- [DiFFoZTweaks](https://thunderstore.io/c/lethal-company/p/DiFFoZ/DiFFoZTweaks/) is recommended to fix SoftMask error instead of EnemySkinRegistry's "SoftMaskKiller" since it causes HUD stuff to look weird.
