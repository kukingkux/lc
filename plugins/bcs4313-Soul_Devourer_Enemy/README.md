# Soul Devourer

The secret enemy from Moai Enemy is finally here as a standalone mod!
Adds a disgusting skeleton giant to tear you limb from limb! It spawns at night on all moons.

<img src="https://cdn.allthepics.net/images/2025/03/29/Screenshot-2025-03-29-201309.png" width="200" height="200"/>&nbsp;&nbsp;&nbsp;&nbsp;

This mod is fully customizable. You can customize the monster's:
- spawn rate
- if it spawns in the day
- speed
- general size (affects pitch as well)
- size variant chance
- volume

<br>
This enemy may spawn as a size variant, making it smaller and/or larger
than usual (20% chance).
