# Deepcore Mines

Welcome to the deepcore mines, a 5 story mine interior meant to act as a mineshaft variant. It features an elevator that goes from bottom to top floor, a ladder used to access intermediate floors and other unique mechanics such as exploding dynamite.

Features unique tag based variants that appear on some moons. See below for screenshots and tag names.

# IMPORTANT NOTES:

The elevator currently only goes to the bottom and top floor! use the LADDER SHAFT to access intermediate floors! 2 Handed items can be taken up and down it.

Warning... Enemies can use the ladder too. You are not safe.

In the future, more tiles and mechanics may be added. The elevator will also be upgraded to visit all 5 floors.
