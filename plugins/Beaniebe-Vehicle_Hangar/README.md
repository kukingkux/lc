# Vehicle Hangar

The vehicle hangar is a large interior that takes place in a vehicle storage facility. It has fans that can be used to float the player through the air, custom apparatus events and even 3 landmine variants.


If you have any bug reports send them to beaniebe on discord!

It has 3 different variants that occur based on level tags (moon tags):
