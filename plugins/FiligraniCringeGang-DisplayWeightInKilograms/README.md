# Display Weight In Kilograms

This mod converts pounds (lb) of carrying weight to kilograms.

#Don't get fooled by the game:
- Some items has literally zero weight, and even in kilograms will display zero.
- Most of the items have very wacky weights, like big bolt is around 8 kilograms, however, all the convertations are calculated correctly.
- Game has hardcoded weight display limit of 4762.7kg. In normal gameplay you will not be able to reach that weight, however while using some odd mods, keep that in mind.
