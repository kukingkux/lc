# JMC ScanVan

(V81) The JcJenson Motor Division, in co-partnership with Vandal Electronics are proud to announce, the all new for 2428, ScanVan!
This Lethal Company mod adds a 'new' vehicle to the game: The ScanVan! to the terminal for `440` credits. It is a Cruiser based, 2-door luton-style van with an enclosed roof, with plenty extra features for you great little company assets! an ideal alternative to the Company Cruiser, some may say.

## Feedback

You can find me and my little old thread via the Lethal Company Modding Discord, or you can use the attached GitHub link if you need to report an issue, etc.

## Features
* Three seats
* Manual gearbox with clutch system
* Enclosed roof, protecting you from various outdoor foes.
* Functonal gauge cluster (speedometer, tachometer, dashboard warning lights, etc)
* Functional center and side view mirrors
* A functional live radio system (RadioFurniture in your truck!) (Disabled by default, disable the DMCA-Music config to enable this)
* Fun interactables such as electric windows, sunvisors, hazard warning lights, etc!
* **10** default radio tracks (DMCA-free)
* Service manual courtesy of Glojam, who made the Company Hauler mod
* Functional climate control (LethalElements compatability (LE does not work in V80 as of writing.. but when it is fixed this should still work!))
* ...other things?

## Controls

The default control scheme is as follows:

 - W = Accelerate
 - S = Brake 
 - A = Steer Left
 - D = Steer Right
 - SpaceBar = Rock
 - LeftShift = Clutch
 - Mouse Wheel Up = Shift Up 
 - Mouse Wheel Down - Shift Down
 
## Compatibility
* Should mostly be compatible with any other vehicle mods (i.e. Company Hauler)

## Not compatible, but should not break or cause issues if installed:
- LategameCompanyCruiserUpgrades
- CruiserImproved
- BetterVehicleControls
- CruiserAdditions
- CompanyCruiserSteeringFix (wouldn't even be needed anyways)
- BetterCruiserSync
- 2 Story RV Cruiser
- YPlayBoombox

Some mods listed may have partial or full self-implementation, such as BetterVehicleControls keybinds, or configs such as automatically centering steering wheel.

## Mods that will cause issues/do not work with the ScanVan:
- CruiserSkin
- CruiserMap
- LCVR (this will be implemented in the future)

### Compatible:
- LethalElements (custom support)
- CruiserTunes (You must disable the DMCA-Music config in-order to play your own music.)
- ImmersiveVisor
- LethalMin
