# Better Spray Paint v80 Fix

This mod fixes an issue with the Better Spray Paint mod (by taffyko) where if the spray can is in the utility slot, many features will not work properly, such as:

* Ability to erase
* Ability to switch color
* Ability to change size
* Size HUD



The fixes from this mod are client-side, the host only needs the Better Spray Paint mod for you to use this mod.



### Compatibility

Hasn't been tested with any other mods, don't see a reason why it would conflict.
