- An inpenetrable fog rolls in, strange noises fill the air, Eyes pearing from the darkness. 

- Black fog is a weather revolving around a rare event of the same name from Voices of the Void, with similar function and execution. A dark fog rolls in, reducing the Players view distance severely, prying eyes watching you. The fog follows you indoors, and unlike regular foggy weather, it's only temporary, disappearing around 3 PM.
