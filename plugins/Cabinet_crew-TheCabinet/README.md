# The Cabinet

### The Cabinet is a standalone interior entity with multiple unique interactions to be discovered!

### It can be found commonly in Mansions, hiding amongst the furniture...

-----------------

### How it works

<details>

<summary>(SPOILERS)</summary>


The Cabinet is a passive entity that only wants to watch from the shadows. 
<br/>
<br/>When spawned it hides within its shell, waiting for an employee to walk by. <br/>
If not looked at, it will emerge and begin following. <br/>
Once it catches up, it will quickly hide inside once more.
<br/>
<br/>However, should someone witness what lies within, the cabinet will quickly go berserk and maim all who have seen it!
<br/>
<br/>Prospective employees may be able to utilize the cabinet as a portable storage unit. <br/>
Just be sure not to overdo it or you'll have a mess to clean up.

</details>

## How to install

- Use mod manager for the easiest installation. 

OR

- Make sure you have [BepInEx](https://thunderstore.io/c/lethal-company/p/BepInEx/BepInExPack/) and [LethalLib](https://thunderstore.io/c/lethal-company/p/Evaisa/LethalLib/) installed.
- Download this mod and extract it. 
- Move folder called "TheCabinet" into (`\GAME_LOCATION\Lethal Company\BepInEx\plugins`)
