# Revisit Stingers
The interiors in Lethal Company each have a unique "stinger" track that plays the first time you enter it:
- Factory https://youtu.be/aDB-aKnUnQ4
- Manor https://youtu.be/yl4OFkyLaaE
- Mineshaft https://youtu.be/bKsuYiF8sw4

These stingers only play one time <b>ever</b> unless you completely reset your save file, which is a shame, because they are quite atmospheric.

In vanilla, these stingers also don't play when using the inverse teleporter to enter the building. (Instead, they play if you exit the building and then re-enter via one of the entrance doors.) I have fixed both of these issues.

This is a simple plugin that will allow you to define additional criteria for the stingers to play:
- By default, this mod will play the stingers any time you encounter an interior with a 25% chance or less of appearing on that specific moon. (Manor on Titan, Factory on Artifice, Mineshaft on Rend, etc.)
  - The exact percentage can be changed - use the wiki to help determine a moon's specific interior spawn rates.
- You can also set a percentage chance for a stinger to play when you interrupt an ambient music track by entering the building.
- If you want, you can also set a flat rate for the stinger to play, even when neither of the above conditions are met.
