# Lethal Casino Tweaks

Patches the [LethalCasino](https://thunderstore.io/c/lethal-company/p/mrgrm7/LethalCasino/) mod to add a few features some friends and I wanted.

## Features

The current features are all based around making the Blackjack game more realistic/enjoyable.

1. The Blackjack game only reshuffles the deck when necessary (deck is empty or upon playing the first game). This allows for players to try counting cards.
2. Modifies the shuffling so that it only shuffles back cards that are out of play, not ones on the table.
3. Adds a "double down" mechanic, inspired by the [Double Down in Lethal Casino](https://thunderstore.io/c/lethal-company/p/Cardboard_Box/Double_Down_in_Lethal_Casino/) mod. It defaults to "U", can be rebound, and shows a hint under the "Hit" button.
4. The deck model on the table is updated to shrink as the number of cards remaining in the deck decreases.
