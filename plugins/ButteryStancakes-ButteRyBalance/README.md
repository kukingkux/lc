# ButteRyBalance

A collection of rebalances for several aspects of the game. <b>Everyone must install</b>, but it is fully configurable and fully networked - the host controls the session's config.

Updating settings in realtime (for example, using [LethalConfig](https://thunderstore.io/c/lethal-company/p/AinaVT/LethalConfig/)) is not guaranteed to be supported. For best results, the host should reboot their game after making any changes to the config.

Be wary of conflicts with other mods, since this makes a lot of changes across the board. The config has been set up so that disabling individual features should prevent any conflicts with mods that do similar things.

# ⚠️ Note About Dependencies

All of the dependencies listed are soft dependencies, and can be removed or disabled if necessary - but certain features do not work without them. <b>Please do not report issues with settings related to the mods marked as dependencies, unless you have those dependencies installed and enabled.</b>

(ex: Barbers' `Dynamic Spawn Settings` *requires* both [Barber Fixes](https://thunderstore.io/c/lethal-company/p/ButteryStancakes/BarberFixes/) and [Spawn Cycle Fixes](https://thunderstore.io/c/lethal-company/p/ButteryStancakes/SpawnCycleFixes/) to function)
