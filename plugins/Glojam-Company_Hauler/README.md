# Company Hauler
This mod adds an new vehicle—the ***Company Hauler***—to the terminal for `530` credits. It is a four door pickup truck with an enclosed roof.

Note: this is *NOT* a cruiser model replacement mod. It adds a whole new vehicle to the terminal, courtesy of LLL!
Speaking of which, this should be compatible with all vehicle mods that also use LLL.

## Features
* Five seats
* Vanilla-style design
* Enclosed roof - protects against various outdoor enemies
* Working tachometer, speedometer, and warning lights
* Side mirrors
* Fun interactables such as automatic windows, a sunroof, and a tailgate!
* More than **3x** the health of the Cruiser, *at the expense of having no boost*
* **8** radio songs
* Custom sounds, driver animations, and manual
* Other things...?

## Compatibility
- Should be compatible with all other vehicle mods that depend on LLL
- Not compatible with pre-V80+ modpacks

## Not compatible, but should not break or cause issues if installed:
- LategameCompanyCruiserUpgrades
- CruiserImproved
- BetterVehicleControls
- CruiserAdditions
- CompanyCruiserSteeringFix (wouldn't even be needed anyways)
- BetterCruiserSync
- 2 Story RV Cruiser
- YPlayBoombox

## Mods that will cause issues/do not work with the Hauler:
- CruiserSkin
- CruiserMap
- LCVR

### Compatible:
- CruiserTunes 
- ImmersiveVisor
- LethalMin
