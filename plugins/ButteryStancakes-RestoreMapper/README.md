# Restore Mapper
Adds the unused Mapper item to the store.

Functionality remains more or less identical to vanilla, but with some fixes:
- Mapper always follows the player holding the device, instead of who is selected on the ship radar
- Multiple mappers can display different things on their screens
- Mapper now plays a short animation when updating the screen for additional feedback

Some added perks:
- Mapper can see inside of caverns, where the ship radar would normally display a "No signal!" screen.
- Mapper draws a line back to main entrance for its user
  - This must be enabled in the config (seems like [Zeekerss intended this feature as a compromise for the "screen duty" player](https://www.patreon.com/posts/v70-130558963))
- Mapper works in utility slot as of v80+!

Inventory icon was generated by MiraakThuri using [RuntimeIcons](https://thunderstore.io/c/lethal-company/p/LethalCompanyModding/RuntimeIcons/).
