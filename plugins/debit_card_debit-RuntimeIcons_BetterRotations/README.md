# Supported mods:
> The difference between "rotation" and "image" here is that the former are coordinates that tell RuntimeIcons how each item will be rotated when the icon is generated. The latter are direct `.png` replacements for when an item already has a special icon that can't be replaced with a normal rotation, or has additional edits made.

#### If you wish to disable the new equipment icons:
In the config file `com.github.lethalcompanymodding.runtimeicons.cfg`, make sure "List Behaviour" is set to `BlackList`, then in the "Item List" field, enter the desired path/items.

Below is the list of tools you can copy/paste to disable them all:

> `Vanilla/Gift, Vanilla/Pro-flashlight, Vanilla/Lockpicker, Vanilla/Shovel, Vanilla/Flashlight, Vanilla/Walkie-talkie, Vanilla/Key, Vanilla/Shotgun, Vanilla/Stun grenade, Vanilla/TZP-Inhalant, Vanilla/Weed killer, Vanilla/Zap gun, Vanilla/Boombox, Unknown/clipboard, Vanilla/Radar-booster, Vanilla/Jetpack, Vanilla/Belt bag, Vanilla/Extension ladder, Vanilla/Spray paint, Vanilla/Binoculars`

#### HOW TO USE ARACHNOPHOBIA MODE IMAGE: 

<details>
<summary>Click to expand</summary>

> Make sure you have [SellBodiesArachnophobiaSupport](https://thunderstore.io/c/lethal-company/p/NotezyTeam/SellBodiesArachnophobiaSupport/) installed

> Navigate to `[YourProfile]/BepInEx/plugins/RuntimeIcons/png/15sellbodiesfixed/`, rename or delete `dead bunker spider.png` and then rename `dead bunker spider arachnophobia.png` to `dead bunker spider.png` (Be sure the filename is all lowercase). The image replacement should take affect next time the game is launched (however, I've found that even if you load a save file with Arachnophobia mode turned on, any existing dead bunker spiders on the ship will retain the original model, so be aware of that. Newly spawned-in spiders should be fine.)

</details>
